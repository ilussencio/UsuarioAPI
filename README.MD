### DETALHAMENTO
Rota <i>/usuario</i> metodo <i>`POST`</i>

Objetivo: Criar um usuario

Envio:
```json
{
	"nome":"Nome completo",
	"telefone":"34912345678",
	"email":"email@email.com",
	"username":"username",
	"senha":"password"
}
```
Retorno:
```json
{
  "id": 2,
  "nome": "Nome completo",
  "telefone": "34912345678",
  "email": "email@email.com",
  "username": "username",
  "senha": "password",
  "nivelAccess": "USER",
  "createAt": "2022-10-25T17:41:29.1127358",
  "updateAt": "2022-10-25T17:41:29.1127358"
}
```
