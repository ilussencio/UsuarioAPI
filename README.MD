### DETALHAMENTO
Rota <i>/usuario</i> metodo <i>`POST`</i>

Objetivo: Criar um usuario

Envio:
```json
{
	"nome":"Nome do usuario",
	"telefone":"34999999999",
	"email":"email@email.com",
	"usuario":"usuario",
	"senha":"senha"
}
```
Retorno:
```json
{
	"id": 1,
	"nome": "Nome do usuario",
	"telefone": "34999999999",
	"email": "email@email.com",
	"usuario": "usuario",
	"senha": "senha",
	"createAt": "2022-10-19T23:33:20.148684",
	"updateAt": "2022-10-19T23:33:20.148813"
}
```
